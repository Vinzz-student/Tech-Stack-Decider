# Tech Stack Decider

Aplikasi web interaktif untuk membantu developer pemula memilih tech stack yang tepat untuk project mereka.

## Fitur

- ✅ **Questionnaire Interaktif**: Jawab pertanyaan tentang project, tim, dan budget
- ✅ **Rekomendasi Personalized**: Dapatkan rekomendasi stack berdasarkan kebutuhan
- ✅ **Stack Comparison**: Bandingkan dua tech stack secara detail
- ✅ **Real Project Examples**: Lihat contoh project nyata dengan berbagai stack
- ✅ **Learning Curve Estimator**: Perkiraan waktu dan effort untuk mempelajari stack
- ✅ **Responsive Design**: Bekerja sempurna di desktop dan mobile

## Tech Stack yang Direkomendasikan

- **MERN Stack**: MongoDB, Express.js, React, Node.js
- **JAMstack**: JavaScript, APIs, Markup
- **LAMP Stack**: Linux, Apache, MySQL, PHP
